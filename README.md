Instalación

1. Clona el repositorio:
git clone https://github.com/tu-usuario/tu-repo.git

2. Instala las dependencias:
pip install -r requirements.txt

3. Ejecuta la aplicación:
   python app.py


---

**Nota sobre las dependencias:** El paquete `user-agents` en pip lleva guion, pero en Python se importa con guion bajo (`from user_agents import parse`). El comando `pip install` que te di ya lo maneja correctamente.
